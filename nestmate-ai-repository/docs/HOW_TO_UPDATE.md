# How to Update NestMate AI
### (No coding knowledge needed — step by step)

## PART 1 — Setting Up GitHub (one time only)

**Step 1 — Create a GitHub account**
1. Go to github.com
2. Click Sign up
3. Choose a username, enter email and password
4. Verify your email

**Step 2 — Create a new repository**
1. Click the + button top right
2. Click New repository
3. Name it: nestmate-ai
4. Set to Public
5. Click Create repository

**Step 3 — Upload your files**
1. Click "uploading an existing file"
2. Drag ALL files keeping same folder structure
3. Write "first upload" in the message box
4. Click Commit changes

---

## PART 2 — Deploy Free on Vercel

1. Go to vercel.com
2. Click Sign up → Continue with GitHub
3. Click Add New Project
4. Find nestmate-ai → click Import
5. Click Deploy
6. Wait 1 minute — your site is LIVE!
   URL will be: nestmate-ai.vercel.app

---

## PART 3 — Updating in Future

Edit directly on GitHub:
1. Go to your repo on github.com
2. Click the file you want to change
3. Click the pencil icon (edit)
4. Make changes
5. Click Commit changes
6. Vercel updates your live site in 1 minute automatically!

---

## PART 4 — Common Changes

ADD new chat answer → edit src/responses.js
CHANGE colours → edit src/style.css (find :root at top)
CHANGE heading text → edit index.html
ADD feature card → copy a feature-card block in index.html

---

## PART 5 — Custom Domain (Optional, ~10 per year)

1. Buy domain at namecheap.com
2. In Vercel: Settings → Domains → Add your domain
